HR & Workforce Analytics — Power BI Build Guide

Use the Excel file Dashboard2_HR_Workforce_Data.xlsx alongside this guide. Follow each step in order.

1
Load the Data into Power BI
⏱ ~15 min

File to use: Dashboard2_HR_Workforce_Data.xlsx — contains 4 sheets covering 214 simulated employees across 8 departments.

  1. Open Power BI Desktop → Get Data → Excel Workbook
  2. Select the file → tick all 4 sheets: Employee_Master · Headcount_Attrition · Performance_Reviews · Recruitment_Pipeline
  3. Click Transform Data → verify first row used as headers in each table
  4. Set data types: Date columns → Date · Salary/Bonus → Decimal Number · IDs → Text · Rate columns (Attrition_Rate_Pct, Offer_Acceptance_Rate) → Percentage
  5. Click Close & Apply
✅ The Attrition_Rate_Pct column contains Excel formulas (e.g. =F5/D5). Power Query reads these as decimal numbers — verify they show as 0.11 not 11
2
Build the Data Model
⏱ ~15 min

Go to Model view. Create these relationships:

  • Employee_Master[Employee_ID] → Performance_Reviews[Employee_ID] (One-to-Many)
  • Employee_Master[Department] → Headcount_Attrition[Department] (Many-to-Many)
  • Employee_Master[Department] → Recruitment_Pipeline[Department] (Many-to-Many)

Create a Date Table for time intelligence — in the Home tab, click New Table and paste:

DateTable = CALENDAR(DATE(2022,1,1), DATE(2024,12,31))

Then relate DateTable[Date] → Employee_Master[Hire_Date]

✅ Mark the DateTable as a Date Table: right-click it in the Fields pane → Mark as Date Table → Date column: Date
3
Create DAX Measures
⏱ ~25 min

Right-click Employee_Master in the Fields pane → New Measure. Create each one below:

Total Headcount
Total Headcount = COUNTROWS(Employee_Master)
Active Employees
Active Employees = CALCULATE( COUNTROWS(Employee_Master), Employee_Master[Employment_Status] = "Active" )
Attrition Rate %
Attrition Rate % = DIVIDE( CALCULATE( COUNTROWS(Employee_Master), Employee_Master[Employment_Status] <> "Active" ), [Total Headcount], 0 )
Avg Salary
Avg Salary = AVERAGE(Employee_Master[Salary_USD])
Avg Performance Score
Avg Performance Score = AVERAGE(Performance_Reviews[Performance_Score])
Avg Days to Fill
Avg Days to Fill = AVERAGE(Recruitment_Pipeline[Days_to_Fill])
Promotion Eligible Count
Promotion Eligible = CALCULATE( COUNTROWS(Performance_Reviews), Performance_Reviews[Promotion_Eligible] = "Yes" )
Offer Acceptance Rate
Offer Acceptance Rate = DIVIDE( SUM(Recruitment_Pipeline[Offers_Accepted]), SUM(Recruitment_Pipeline[Offers_Made]), 0 )
✅ Format Attrition Rate % and Offer Acceptance Rate as Percentage · 1 decimal place. Format Avg Salary as Currency · 0 decimals.
4
Design the 3-Page Layout
⏱ ~15 min

Add and rename 3 pages (right-click the page tab):

  • Page 1 — Workforce Overview
  • Page 2 — Attrition & Retention
  • Page 3 — Recruitment & Performance

Canvas background: Format → Canvas background → Color → #F0F4F8

Header bar: Insert → Rectangle → fill #2C5F2E → full width · 55px height. Text box on top: white · 16pt bold.

6
Add Slicers & Cross-filtering
⏱ ~15 min

Add these slicers and sync them across all 3 pages (View → Sync Slicers):

  • Department → Style: Dropdown
  • Year → Style: Tile (shows 2022 · 2023 · 2024)
  • Location → Style: Dropdown
  • Seniority_Level → Style: Checkbox list
  • Employment_Status → Style: Checkbox list
✅ Cross-filtering is automatic in Power BI — click any bar in the department chart and all other visuals on the page filter instantly
⭐ Bonus — Bookmarks: Create two saved states using View → Bookmarks → Add Bookmark. One for "Attrition Focus" (attrition slicer active) and one for "Performance Focus". Add navigation buttons so viewers can toggle between them. This is an advanced feature that seriously impresses interviewers.
7
Format & Polish
⏱ ~20 min
  • Colour palette: #2C5F2E · #52b052 · #28a745 · #dc3545 · #fd7e14 only
  • Font: Segoe UI throughout — consistent and professional
  • Visual borders: Format → Visual border → OFF for all visuals
  • Shadows: Format → Shadow → ON (subtle 2px offset)
  • Grid lines: OFF on all bar charts
  • Mobile layout: View → Mobile layout — stack KPIs vertically for a polished LinkedIn video demo
  • Tooltips: Enable on all charts for richer hover interactions
  • Format Painter: Copy formatting from one visual to all others with the paint roller icon
💡 Once complete, publish to Power BI Service (free account at app.powerbi.com) and get a shareable link to add to your CV, LinkedIn, and GitHub README
