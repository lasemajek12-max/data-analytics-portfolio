Sales & Revenue Analytics — Power BI Build Guide

Use the Excel file Dashboard1_Sales_Revenue_Data.xlsx alongside this guide. Follow each step in order.

1
Load the Data into Power BI
⏱ ~15 min

Download Dashboard1_Sales_Revenue_Data.xlsx from this repository. It contains 4 sheets with 1,200 simulated forex transactions.

  1. Open Power BI Desktop (free at powerbi.microsoft.com)
  2. Click Get Data → Excel Workbook → select the file
  3. In the Navigator, tick all 4 sheets: Sales_Transactions · Monthly_Targets · Client_Retention · Sales_Rep_KPIs
  4. Click Transform Data to open Power Query Editor
  5. For each table, verify Home → Use First Row as Headers is applied
  6. Set column types: Date → Date · Revenue/Cost → Decimal Number · IDs → Text
  7. Click Close & Apply
✅ If the Date column shows as text, click the column header type icon and select Date
2
Build the Data Model (Relationships)
⏱ ~10 min

Click the Model view icon (3rd icon on the left sidebar). Create these relationships by dragging fields between tables:

  • Sales_Transactions[Sales_Rep] → Sales_Rep_KPIs[Sales_Rep]
  • Sales_Transactions[Month] → Monthly_Targets[Month]
  • Sales_Transactions[Client_ID] → Client_Retention[Client_ID]
✅ To create a relationship: drag a field from one table and drop it onto the matching field in another table
⚠️ Keep all relationship directions as Single (not Both) to avoid circular dependency errors
3
Create DAX Measures
⏱ ~20 min

In Report view, right-click Sales_Transactions table in the Fields pane → New Measure. Create each measure below, then press Enter.

Total Revenue
Total Revenue = SUM(Sales_Transactions[Revenue_USD])
Total Profit
Total Profit = SUM(Sales_Transactions[Profit_USD])
Profit Margin %
Profit Margin % = DIVIDE([Total Profit], [Total Revenue], 0)
Total Deals
Total Deals = COUNTROWS(Sales_Transactions)
Avg Deal Size
Avg Deal Size = DIVIDE([Total Revenue], [Total Deals], 0)
Target Attainment %
Target Attainment % = DIVIDE( SUM(Monthly_Targets[Actual_USD]), SUM(Monthly_Targets[Target_USD]), 0 )
Revenue vs Target Variance
Revenue vs Target = SUMX( Monthly_Targets, Monthly_Targets[Actual_USD] - Monthly_Targets[Target_USD] )
✅ Format currency measures: select the measure → Measure Tools tab → Format: Currency · 0 decimal places
Format Profit Margin % and Target Attainment % as Percentage · 1 decimal place
4
Design the Page Layout (3 Pages)
⏱ ~15 min

Right-click the page tab at the bottom to add and rename pages:

  • Page 1 — Executive Overview
  • Page 2 — Sales Rep Performance
  • Page 3 — Client & Product Deep Dive

Set canvas background: Format → Canvas background → Color → enter #F0F4F8

Add a header: Insert → Shapes → Rectangle → fill color #1F4E79, full width, ~55px height. Then Insert → Text Box on top: white text, 16pt bold.

6
Add Slicers & Filters
⏱ ~15 min

Slicers let viewers interactively filter the whole report. Insert these on Page 1:

  • Insert → Slicer → Field: Year → Style: Tile
  • Insert → Slicer → Field: Quarter → Style: Tile
  • Insert → Slicer → Field: Region → Style: Dropdown
  • Insert → Slicer → Field: Client_Segment → Style: Dropdown
  • Insert → Slicer → Field: Product → Style: Dropdown

To sync slicers across all 3 pages: View → Sync Slicers → enable all pages for each slicer.

✅ Style slicers to match: Format → Slicer header → Background #1F4E79 · Text color white · Font size 11pt
7
Format & Polish
⏱ ~20 min
  • Font: Set all visuals to Segoe UI — consistent, professional
  • Colour palette: Stick to #1F4E79 · #2980b9 · #28a745 · #dc3545 · #fd7e14 only
  • Borders: All visuals → Format → Visual border → OFF
  • Shadows: Format → Shadow → ON (2px offset, subtle)
  • Grid lines: Turn OFF x-axis and y-axis gridlines on bar charts
  • Titles: Every visual needs a clear bold title in dark blue
  • Spacing: Leave ~12px gap between every card
  • Format Painter: Use the paint roller icon in the toolbar to copy formatting from one visual to all others instantly
💡 Bonus: Add Tooltips to all charts. Hover-on-hover data is very impressive during live demo presentations to interviewers.
